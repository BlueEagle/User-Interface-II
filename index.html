I've been writing code since I was little. In fifth grade I learned about CheatEngine, a program that allows you to change values in flash games. I thought the idea that programs are made up full of customizable values and was fascinating! Later I learned that there were javascript snippets that would allow me to change the words on a webpage. This was computer magic to me. I needed to learn how to make these awesome things, to become a magician myself. And so my journey began!
